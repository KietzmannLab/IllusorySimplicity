# Plots for Inter-area project

Plots for final results are created from jupyter notebooks in `./notebooks/figs`.

## ERPs

ERPs are computed and plotted in `./notebooks/figs/erp_panels.ipynb`.
Note: ERPs are computed directly by this notebook. This requires a large amount of memory.

## Reliability

Reliability figures are generated in the notebook `./notebooks/figs/electrode_reliability_figs.ipynb`.

## Decoding

ROI-wise decoding results are plotted in the notebook `./notebooks/figs/decoding_perf_per_roi.ipynb`.

Array-wise decoding results are plotted in the notebook `./notebooks/figs/decoding_perf_per_array.ipynb`.

## Inter-area Analyses

### Surfaces

Inter-area fit surfaces are plotted in the notebook `./notebooks/figs/surface_plots_and_average.ipynb`. These include:

- surface plots averaged over all target arrays in one ROI
